# C++ Method Wrapper

Проект для создания системы команд, позволяющей регистрировать методы классов и вызывать их по имени с передачей аргументов через `std::unordered_map`.


## Установка зависимостей

```bash
sudo apt update
sudo apt install libgtest-dev
```

## Ограничения

- Поддерживаются только типы аргументов: `int`, `double` по условию
- Типы возвращаемых значений: `int`, `double`, `void` 

## Быстрый старт

```bash
# Сборка проекта
make

# Запуск тестов
make test

# Запуск демонстрации
make run

# Запуск всего
make check
```

## Основные компоненты

### Wrapper<Class, Ret, FuncArgs...>
Шаблонный класс для обёртывания методов класса произвольной сигнатуры.

### Engine
Движок для регистрации и выполнения команд по имени.

### make_wrapper()
Вспомогательная функция для создания обёрток методов с автоматическим выводом типов.


Проект использует **Google Test framework** для unit-тестирования.
